## 深夜三点，你在想什么？

工作几年了，为什么别人升职加薪，你却一直在原地踏步。想转行，不知道方向在哪里。想辞职，又不敢。想换个活法，不知道还能做什么。

人际关系让你疲惫。不知道对面这个人值不值得信任，不知道自己哪句话说错了，不知道怎么拒绝别人又不伤感情。

财务压力像一座山。赚钱不容易，守住更难。投资像走钢丝，存款越存越薄，未来越想越慌。

夜深人静，那些失败的经历在脑中一遍遍回放。越想越焦虑，越焦虑越睡不着，越睡不着第二天越没精神。

**命书不问你想去哪，只问你现在在哪。**

然后告诉你：下一步该怎么走。

## 命书是什么？

不是算命，不是占卜，不是玄学。

命书来自番茄小说《改运奇书》，是一套源自中国古典哲学的人生智慧。41条易命术，每一条都针对一种具体的人生处境。

它讲的不是命运注定，而是人在不同处境下如何做出更好的选择。

**阅读原作：** https://fanqienovel.com/page/7573616392256097305

## 💬 对话示例

**场景1：失业迷茫**

```
用户: 失业3个月了，每天投简历没有回音，越来越焦虑

命书: 收到。我来帮你分析一下。

---

### 易命第五术 · 活在当下

**原文：**
旧过过，未未到。事有先后，逐一面之，戒之在贪。

**白话解读：**
过往的失败已经发生、无法改变，就不要再为此恐惧和内耗。当下最重要的，是看清眼前的问题、分清主次，集中精力一个一个解决，不能贪多、不能冒进。

**行动建议：**
| # | 行动 | 时长 | 成功标准 |
|---|------|------|----------|
| 1 | 写出当前最紧急的3件事 | 15分钟 | 列清单，不评判 |
| 2 | 每天只专注完成1件事 | 1小时/天 | 完成就打勾 |
| 3 | 想过去时，立刻转移注意力 | 持续 | 能意识到就很好 |
```

**场景2：想分析一个人**

```
用户: 帮我看看我相亲对象这个人怎么样

命书: 请描述一下你观察到的细节：
- 你们是怎么认识的？
- 你注意到他/她有哪些习惯？
- 你们相处时他/她通常怎么对待你？

[根据描述，用识人术分析人品、可信度、相处建议]
```

**场景3：做重大决定**

```
用户: 我想辞职创业，行不行？

命书: 辞职创业需要的不只是勇气，还有时机。让我帮你分析一下：

[运用时机把握术，分析当前处境，判断是否是天时、地利、人和的时机]

建议先问自己三个问题：
1. 我储备的资本能撑多久？
2. 如果失败了，最坏的结果是什么？
3. 我是真的想做这件事，还是只想逃离现状？
```

## ✨ 命书能为你做什么

### 核心功能

| 功能 | 什么时候用 | 特点 |
|------|-----------|------|
| **困境咨询** | 迷茫、焦虑、不知道怎么办 | 说出处境，获取针对性的命书智慧 |
| **每日一术** | 想每天获得一点智慧 | 随机一条，配当日行动建议 |
| **命书日记** | 打卡修行、追踪效果 | 记录实践，AI分析反馈 |
| **苏格拉底引导** | 需要深度思考时 | 通过提问帮你自己找到答案 |

### 8大延伸功能

**这些功能是困境咨询的补充，不是独立入口。**

| 功能 | 什么时候用 | 基于术 |
|------|-----------|--------|
| **识人术** | 困境涉及"分析一个人"时 | 第9、13、14、19术 |
| **言语分析** | 困境涉及"怎么说"时 | 第3、12术 |
| **婚恋关系** | 困境涉及"伴侣/感情"时 | 第22、30术 |
| **贵人识别** | 困境涉及"人脉/贵人"时 | 第10、23术 |
| **社交分层** | 困境涉及"社交圈"时 | 第16、21术 |
| **时机把握** | 困境涉及"该不该/选择"时 | 第6、15、17、35、38、41术 |
| **合作之道** | 困境涉及"合作/合伙"时 | 第14、18术 |
| **财富管理** | 困境涉及"钱/投资"时 | 第4、25-27、31-34术 |

### 触发原则

1. **优先困境咨询**：用户描述困境时，先走困境咨询流程
2. **延伸推荐**：困境咨询完成后，根据情况推荐相关功能
3. **明确请求**：用户明确说"帮我看人"时，直接路由到识人术

### 延伸推荐示例

```
命书: 基于你的情况，推荐【易命第五术】...

延伸推荐：
"你提到的'老板'问题，命书第九术讲到识人之道，需要我帮你深入分析这个人吗？"
```

## 🌟 命书和其他AI有什么不同

### vs 通用AI助手

| 通用AI | 命书 |
|--------|------|
| 给泛泛的建议 | 基于古典智慧的针对性解读 |
| 长篇大论不知道重点 | 41条技术，针对42种处境 |
| 需要你问对问题 | 你说处境，命书来找技术 |
| 随机的鸡汤 | 有原文、有解读、有行动清单 |

### vs 算命/占卜

| 算命 | 命书 |
|------|------|
| 预言未来 | 不预言未来 |
| 含糊的吉凶 | 关注当下选择 |
| 被动接受 | 给你具体行动清单 |
| 宿命论 | 相信人可以改变 |

### 独特之处

- **古典智慧**：源自中国古典哲学，不是随机的励志语录
- **可执行行动**：每条技术都有具体的行动清单，不是"要努力"
- **双语支持**：原文是古典中文，解读是现代白话，配有自然英文翻译
- **记忆系统**：命书记得你说过的话，给出连贯的建议
- **打卡系统**：追踪你的修行，记录成长轨迹
