# 命书修行提醒模板

## 晨间与晚间提醒模板

早晨提醒帮助用户开启新一天的修行，晚间提醒帮助用户回顾与沉淀。

---

## 模板一：晨间提醒 (Morning Reminder)

### 触发条件

当用户说「早安」「早上好」「新的一天」「开始修行」时，或系统检测到早晨时段（5:00-12:00）时触发。

### 模板内容

```
☀️ 早安，{username}

今天是 {date}。
你已经连续修行 {streak} 天了 {'🔥' if streak >= 7 else '✨' if streak >= 3 else ''}

最佳记录：{best_streak} 天

━━━━━━━━━━━━━━━━

今日推荐修行：

【{technique_number}术 · {technique_name}】

{technique_excerpt}

{technique_translation}

━━━━━━━━━━━━━━━━

今日行动建议：

| # | 行动 | 时长 | 说明 |
|---|------|------|------|
| 1 | {action_1} | {duration_1} | {action_1_desc} |
| 2 | {action_2} | {duration_2} | {action_2_desc} |
| 3 | {action_3} | {duration_3} | {action_3_desc} |

━━━━━━━━━━━━━━━━

今日提示：

"{daily_quote}"

有什么想在今天特别留意的吗？
```

### 晨间激励语

根据连续天数显示不同激励：

| 连续天数 | 激励语 |
|----------|--------|
| 1天 | 「新的一天，新的开始」 |
| 2-3天 | 「习惯在萌芽，继续浇水」 |
| 4-6天 | 「你已经走了一段路，继续」 |
| 7-14天 | 「一周的修行，力量在累积」 |
| 15-29天 | 「内在的变化在悄然发生」 |
| 30天以上 | 「修行已成为你的一部分」 |

### 变量说明

| 变量 | 说明 | 示例 |
|------|------|------|
| {username} | 用户昵称，未知则显示「朋友」 | 小明 |
| {date} | 当日日期 | 2026-04-28 |
| {streak} | 当前连续天数 | 5 |
| {best_streak} | 历史最长连续天数 | 12 |
| {technique_number} | 技术编号 | 第12术 |
| {technique_name} | 技术名称 | 顺势而为 |
| {technique_excerpt} | 技术核心原文或解读 | 顺势而为，待时而动... |
| {technique_translation} | 技术英文翻译（可选） | Go with the flow, wait for the right moment... |
| {action_1/2/3} | 推荐行动1/2/3 | 观察今天遇到的「势」 |
| {duration_1/2/3} | 行动时长 | 10分钟 |
| {action_desc} | 行动描述 | 留意身边正在发生的变化 |
| {daily_quote} | 当日引述 | 今日适合「看」而非「做」 |

## 模板二：晚间提醒 (Evening Reminder)

### 触发条件

当用户说「晚安」「结束今天」「今天总结」时，或系统检测到晚间时段（20:00-次日2:00）时触发。

### 模板内容

```
🌙 晚安，{username}

今天 {date} 的修行记录，等你来整理。

━━━━━━━━━━━━━━━━

📊 今日数据回顾：

| 指标 | 数值 |
|------|------|
| 连续修行 | {streak} 天 |
| 今日评分 | {today_rating}/5 |
| 今日心情 | {mood} |
| 应用技术 | {techniques_used} |

━━━━━━━━━━━━━━━━

💭 今日反思问题：

1. 今天最让你感到挑战的是什么？
   「{challenge_prompt}」

2. 命书智慧如何帮助你应对？
   「{application_prompt}」

3. 明天你想要什么不同的结果？
   「{tomorrow_prompt}」

━━━━━━━━━━━━━━━━

✨ 今日亮点：

{insights}

━━━━━━━━━━━━━━━━

🌅 明日预告：

明日推荐技术：【{tomorrow_technique_number}术 · {tomorrow_technique_name}】

{tomorrow_technique_preview}

明天见。
```

### 晚间数据统计

| 指标 | 说明 |
|------|------|
| {streak} | 当前连续天数 |
| {today_rating} | 今日效果评分（1-5分） |
| {mood} | 今日心情状态描述 |
| {techniques_used} | 今日应用的技术列表 |

### 晚间反思引导

| 问题 | 用途 |
|------|------|
| {challenge_prompt} | 「今天有什么事让你感到压力或困扰吗？」 |
| {application_prompt} | 「今天有没有用到命书的智慧？怎么用的？」 |
| {tomorrow_prompt} | 「如果明天可以有一点不同，你会希望是什么？」 |

### 晚间亮点生成

根据今日记录内容生成亮点描述：

- 评分4-5分：「今天的效果很棒，说明你在对的路上」
- 评分2-3分：「有起伏是正常的，重要的是你没有放弃」
- 尝试新技术：「恭喜你探索了新的技术领域」
- 连续坚持：「能在今天继续修行，本身就是一份成就」

### 晚间结束语

根据连续天数显示不同结束语：

| 连续天数 | 结束语 |
|----------|--------|
| 1-2天 | 「每一天都是新的开始，明天见」 |
| 3-6天 | 「坚持的你在发光，明天见」 |
| 7-13天 | 「修行路上，你不孤独，明天见」 |
| 14-29天 | 「内在的力量在增长，明天见」 |
| 30天以上 | 「修行已成为你的习惯，明天继续」 |

## 与日记系统的集成

### 数据流向

```
晨间提醒 → 触发日间修行记录 → 晚间提醒 → 触发日记整理 → journal_intake.md
```

### 集成点

| 模块 | 说明 |
|------|------|
| journal_manager.py | 提供 streak 数据、推荐技术 |
| journal_intake.md | 用户完成记录后触发反馈 |
| journal_feedback.md | 根据今日数据生成晚间回顾 |
| guidance.md | 反思问题使用苏格拉底式引导 |

### 使用时机

1. **晨间提醒**：每日首次交互时自动检测并发送
2. **晚间提醒**：用户表示要结束时，或检测到晚间时段时发送
3. **跳过逻辑**：如用户今日已记录，直接进入正常对话流程

---

## 注意事项

- 晨间提醒不强制要求回复，让用户自然进入状态
- 晚间提醒如果用户已记录完整，直接使用数据，不再重复询问
- 所有模板都是起点，根据用户反馈逐渐个性化
- 提醒语气温暖但不啰嗦，鼓励但不说教
