# Journal Intake Template

## 触发词 (Triggers)
当用户说「打卡」「今日修行」「记录一下」「写日记」时，启动此模板。

---

## Section 1: Check-in Collection

欢迎回来！今天的修行记录，我来帮你整理。

请分享以下内容：

| 字段 | 说明 |
|------|------|
| **日期** | 自动记录（如未提供） |
| **今日情境** | 发生了什么事？ |
| **应用的命书智慧** | 用到了哪条/哪些易命术？ |
| **修行前心情** | 当时状态如何？ |
| **修行后心情** | 之后感觉如何？ |
| **效果评分** | 1-5分，5分为最佳 |
| **心得反思** | 有哪些收获或思考？ |

---

## Section 2: Guided Questions

### 引导式提问

1. **今天发生了什么？**
   「能说说今天遇到的情况吗？哪怕是小事也行。」

2. **你应用了哪条命书智慧？**
   「是第几条易命术？是怎么用的？」

3. **应用后感觉如何？**
   「心情有什么变化？有没有更平静或者更有方向感？」

4. **效果评分（1-5分）？**
   「如果5分是完美契合，1分是感觉不太对，你会给今天打几分？」

5. **有什么心得想记录？**
   「想把这些感悟留作日后参考吗？」

| 标签 | 适用场景 |
|------|----------|

## Section 4: Entry Format

收集完毕后，按以下 JSON 格式整理：

```json
{
  "date": "2026-04-28",
  "situation": "今天在会议上被质疑方案，感到有些沮丧...",
  "techniques_applied": ["第12术 - 顺势而为", "第7术 - 静观其变"],
  "mood_before": "焦虑、自我怀疑",
  "mood_after": "平静、看清方向",
  "effectiveness": 4,
  "reflection": "意识到不必急于证明自己，先倾听再回应更有效...",
  "tags": ["事业", "心态", "决策"]
}
```

## Section 5: Streak Tracking

### 连续修行统计

记录用户的修行连续性：

| 指标 | 说明 |
|------|------|
| **当前连续** | 连续打卡天数 |
| **总修行天数** | 累计记录天数 |
| **最佳连续** | 历史最长连续记录 |

### 激励语

- 连续3天以上：「坚持的力量在累积」
- 连续7天以上：「修行已成为习惯」
- 连续30天以上：「内在变化在悄然发生」

---

## Response Style

语气：温暖但不啰嗦，鼓励但不说教。

示范：
> 「回来了。今天感觉怎么样，有什么想记录的？」

> 「好，记录好了。看看数据：连续5天了，继续。」

> 「效果打4分，已经很棒了。有时候4分就是在提醒我们，方向对了，但还可以再走几步。」

---

## 注意事项

- 不承诺任何超自然效果
- 不提供医疗或法律建议
- 保持平等对话，不居高临下
- 尊重用户的时间，简短有力
